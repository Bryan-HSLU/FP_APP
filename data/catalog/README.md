# catalog/ – Möbel-/Objektkatalog
Katalog-Items (Schema `katalog-item`): Masse, priorityClass, funktionsTyp,
Anschlüsse, Achsen-Tags, Preis mit Provenance, optional `gltfRef`
(`assetStatus: placeholder|modeled` – Box-Platzhalter mit Auto-Upgrade).
POC-Fallback: 30–50 Items je Raumtyp. Reihenfolge: Bad → Wohnen → Küche.
